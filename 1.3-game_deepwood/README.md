<b>Название:</b> Анализ данных онлайн‑игры «Секреты Темнолесья».

<b>Цель проекта:</b> провести разведочный анализ данных и решить серию аналитических задач на базе данных игры «Секреты Темнолесья», чтобы получить значимые выводы для развития игрового продукта.

<b>Объект анализа: база данных (схема <b>`fantasy`</b>) онлайн‑игры «Секреты Темнолесья»</b>.

<b>Инструменты:</b>
- тренажёр (начальный этап);
- DBeaver (для подключения к БД и выполнения SQL‑запросов);
- MO Word для формирования выводов.

<b>Этапы работы:</b>
- настройка окружения: установка и настройка DBeaver для подключения к базе данных.
- разведочный анализ данных (EDA):
  * изучение структуры таблиц в тренажёре Яндекс Практикума;
  * самостоятельный анализ остальных таблиц через DBeaver;
  * проверка качества данных (пропуски, дубликаты, аномалии), выявление закономерностей.

<b>Решение аналитических задач:</b>
- формулировка и выполнение SQL‑запросов для решения конкретных бизнес‑вопросов (ad-hoc задачи);
- расчёт ключевых метрик, анализ поведения игроков и т. д.

<b>Формулировка выводов:</b> интерпретация результатов запросов, подготовка выводов и рекомендаций на основе данных.
