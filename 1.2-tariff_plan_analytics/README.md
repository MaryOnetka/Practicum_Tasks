Название: Анализ активности клиентов и эффективности тарифных планов «Мегасеть»

Цель проекта: проанализировать использование услуг связи клиентами компании «Мегасеть» по тарифным планам Smart и Ultra, оценить долю клиентов с перерасходом лимитов и подготовить данные для обновления линейки тарифов и разработки спецпредложений.

Бизнес‑задача:

обновить тарифные планы под актуальные запросы клиентов;

разработать систему специальных предложений для самых активных пользователей.

Объект анализа: два тарифных плана компании «Мегасеть»:

Smart: ежемесячный платёж — 550 руб., включает 500 мин звонков, 15 ГБ интернета, 50 сообщений; стоимость сверх лимита: минута — 3 руб., ГБ — 200 руб., сообщение — 3 руб.

Ultra: ежемесячный платёж — 1950 руб., включает 3000 мин звонков, 30 ГБ интернета, 1000 сообщений; стоимость сверх лимита: минута — 1 руб., ГБ — 150 руб., сообщение — 1 руб.

Задачи аналитика:

Выгрузить данные об активности клиентов за календарный месяц: суммарная длительность звонков (с округлением вверх), объём интернет‑трафика, количество сообщений.

Рассчитать текущие траты клиентов с учётом тарифных лимитов и стоимости услуг сверх плана.

Для каждого тарифа рассчитать средние траты клиентов (учитывая только действующих пользователей).

Среди активных клиентов выделить тех, кто пользуется услугами сверх тарифного плана, и посчитать:

средние расходы для каждого тарифа;

среднее значение переплаты для каждого тарифа.

Этапы работы:

Проверка данных: поиск и устранение пропущенных значений и дубликатов.

Подготовка данных: формирование набора данных об активности клиентов (звонки, интернет‑трафик, сообщения).

Анализ и расчёты: написание SQL‑запросов для решения поставленных задач и расчёта ключевых метрик.

Источник данных: база данных с информацией об активности клиентов компании (включая данные о звонках, интернет‑сессиях и сообщениях).
