<b>Название:</b> Анализ развития игровой индустрии (2000–2013 гг.) для продвижения игры «Секреты Темнолесья».

<b>Цель:</b> помочь команде игры «Секреты Темнолесья» подготовить статью для IT‑ресурса о развитии игровой индустрии в начале XXI века — с фокусом на RPG‑игры. Статья должна привлечь аудиторию, интересующуюся старыми играми, и заинтересовать её продуктом команды.

<b>Ключевая тема проекта:<b>
- обзор игровых платформ;
- объёмы продаж игр по жанрам;
- региональные предпочтения игроков;
- акцент на жанре RPG.

<b>Период анализа:</b> 2000–2013 годы.

<b>Источник данных:</b> датасет <b>`/datasets/new_games.csv`</b> с информацией о продажах и оценках игр (пользовательских и экспертных).
|Поле |Описание|
|-----|--------|
|Name |название игры|
|Platform |название платформы|
|Year of Release |год выпуска игры|
|Genre |жанр игры|
|NA sales |продажи в Северной Америке (в миллионах проданных копий)|
|EU sales |продажи в Европе (в миллионах проданных копий)|
|JP sales |продажи в Японии (в миллионах проданных копий)|
|Other sales |продажи в других странах (в миллионах проданных копий)|
|Critic Score |оценка критиков (от 0 до 100)|
|User Score |оценка пользователей (от 0 до 10)|
|Rating |рейтинг организации ESRB (англ. Entertainment Software Rating Board)|

<b>Задачи проекта:</b>
- загрузка и первичное знакомство с данными:
  * загрузить библиотеки Python и датасет;
  * вывести первые строки и результат info();
  * оценить объём данных, типы, пропуски, особенности;
- предобработка данных:
  * привести названия столбцов к snake_case;
  * проверить и преобразовать типы данных;
  * посчитать и обработать пропуски (в т. ч. заменить пропуски в продажах на среднее по платформе и году);
  * найти и устранить явные/неявные дубликаты;
  * нормализовать текстовые значения (регистр и т. д.);
- фильтрация: отобрать данные за 2000–2013 годы, сохранить в отдельный датафрейм;
- категоризация:
  * разделить игры на категории по оценке пользователей (высокая: 8–10; средняя: 3–8; низкая: 0–3);
  * разделить игры на категории по оценке критиков (высокая: 80–100; средняя: 30–80; низкая: 0–30);
  * выделить топ‑7 платформ по количеству выпущенных игр за период.
- формулировка выводов:
  * описать проделанную работу;
  * дать характеристику итогового среза данных и новых полей.

<b>Инструменты:</b> Jupyter Notebook (Python).
