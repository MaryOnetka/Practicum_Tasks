<b>Цель проекта:</b> помочь агентству недвижимости выйти на новый рынок (СПб и ЛО), определив перспективные сегменты и сезонные тенденции для формирования бизнес‑стратегии и планирования маркетинговых кампаний.

<b>Заказчик:</b> агентство недвижимости из Петрозаводска (планирует переезд офиса и выход на рынок СПб и ЛО).

<b>Источник данных:</b> архивная информация сервиса «Яндекс Недвижимость» (схема <b>`real_estate`</b>) — объявления о продаже жилой недвижимости в СПб и ЛО за несколько лет.

<b>Инструменты:</b>
- DBeaver — для подключения к БД и выполнения SQL‑запросов;
- DataLens — для доработки существующего дашборда;
- MO Word для формирования аналитической записки.

<b>Этапы работы:</b>
- знакомство с данными:
  * подключение к базе данных через DBeaver;
  * разведочный анализ данных (изучение структуры, качества, релевантных полей);
- решение аналитических ad hoc‑задач с помощью SQL:
  * задача 1 - определить наиболее привлекательные сегменты недвижимости (СПб и города ЛО) по времени активности объявлений.
  * задача 2 - выявить сезонные тенденции на рынке (периоды повышенной активности продавцов и покупателей) для планирования выхода на рынок и маркетинговых кампаний.
  * сформулировать выводы в виде аналитической записки;
- доработка дашборда в DataLens:
  * доработать существующий дашборд по ТЗ заказчика;
  * добавить новые элементы визуализации на основе проведённого анализа.

<b>Ключевые задачи для анализа:</b>
- сегментирование рынка недвижимости по привлекательности (на основе активности объявлений);
- выявление сезонности спроса и предложения;
- определение оптимальных сроков для публикации объявлений и запуска маркетинга;
- формирование рекомендаций по выбору целевых сегментов недвижимости.
