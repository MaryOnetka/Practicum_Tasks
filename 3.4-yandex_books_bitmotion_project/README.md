<b>Часть 2.</b> Анализ результатов A/B‑теста для BitMotion Kit.

Интернет‑магазин BitMotion Kit тестирует новый интерфейс сайта, чтобы упростить взаимодействие и повысить конверсию зарегистрированных пользователей в покупателей.

<b>Гипотеза теста:</b> упрощение интерфейса приведёт к увеличению конверсии зарегистрированных пользователей в покупателей на ≥ 3 п. п. в течение 7 дней после регистрации.

<b>Параметры теста:</b>
- название: interface_eu_test;
- группы: A (контрольная), B (новый интерфейс).

<b>Задачи:</b>
- загрузить данные:
  * `ab_test_participants.csv` — участники теста (ID пользователя, группа, тест, устройство);
  * `ab_test_events.zip` — события 2020 года (ID пользователя, дата/время события, тип события, детали).
- проверить корректность проведения теста (распределение пользователей, репрезентативность групп и т. д.);
- проанализировать результаты: сравнить конверсию между группами A и B;
- сделать вывод о подтверждении/опровержении гипотезы и дать рекомендации.

<b>Инструменты:</b>
- SQL из тренажёра — для расчёта метрик;
- Jupyter Notebook (Python).
