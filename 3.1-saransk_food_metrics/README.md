<b>Цель:</b> оценить состояние клиентской базы сервиса доставки еды «Всё.из.кафе» в Саранске за май–июнь 2021 года и подготовить практические выводы для принятия бизнес‑решений на основе визуализации ключевых метрик.

<b>Задачи проекта:</b>
- подготовить SQL‑запросы в тренажёре для расчёта ключевых метрик:
  * DAU (daily active users) — количество активных пользователей за день;
  * Conversion Rate — коэффициент конверсии;
  * средний чек — средняя сумма покупки на пользователя;
  * LTV (lifetime value) — совокупная ценность клиента за период;
  * Retention Rate — коэффициент удержания пользователей;
- создать визуализации (графики, диаграммы, таблицы) для каждой метрики — с отражением динамики, закономерностей и аномалий;
- собрать дашборд «Ключевые бизнес‑метрики в Саранске (май — июнь 2021)» с понятными графиками по всем метрикам;
- проанализировать данные и подготовить аналитическую записку с выводами за период с 01.05.2021 по 30.06.2021:
  * динамика DAU (рост/снижение);
  * аномалии в конверсии;
  * изменения среднего чека;
  * динамика Retention Rate в зависимости от периода.

<b>Инструменты:</b>
- SQL из тренажёра — для подготовки запросов и расчёта метрик;
- Yandex DataLens — для визуализации данных и сборки дашборда.

Источники данных:
- `analytics_events` — журнал аналитических событий (визиты, авторизации, заказы и т. д.);
- `advertisement_budgets` — данные о ежедневных рекламных затратах по каналам привлечения;
- `partners` — справочник партнёрских сетей и ресторанов;
- `dishes` — справочник блюд;
- `cities` — справочник населённых пунктов.
